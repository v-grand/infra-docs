# Rozpoczęcie pracy: infra-network

Dokumentacja jest w trakcie tłumaczenia.

Zobacz [wersję angielską](../en/infra-network.md) dla pełnej dokumentacji.

## Krótki opis

**infra-network** zarządza infrastrukturą sieciową, w tym VPC, VPN i siecią mesh Tailscale.

## Główne moduły

- **VPC AWS** - Virtual Private Cloud dla AWS
- **VPC GCP** - Virtual Private Cloud dla GCP
- **VPN** - Konfiguracja VPN typu Site-to-site
- **Tailscale** - Sieć mesh bez konfiguracji
