Документация находится в процессе перевода.

## Краткое описание

**infra-network** управляет сетевой инфраструктурой включая VPC, VPN и Tailscale mesh networking.

## Основные модули

- **VPC AWS** - Virtual Private Cloud для AWS
- **VPC GCP** - Virtual Private Cloud для GCP  
- **VPN** - Site-to-site VPN конфигурация
- **Tailscale** - Zero-config mesh networking
